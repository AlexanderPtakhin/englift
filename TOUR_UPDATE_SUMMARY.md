# Обновление тура EngLift - новые шаги и демо-слово

## 🎯 Что добавлено

### 1. Демо-слово для тура
- **Функция `addDemoWord()`**: Добавляет слово "Hello" в начало списка при старте тура
- **Функция `removeDemoWord()`**: Удаляет демо-слово при завершении тура
- **Характеристики демо-слова**:
  - Слово: "Hello" / "Привет"
  - Транскрипция: `[həˈloʊ]`
  - Уровень: A1
  - Статистика: 3 показа, 2 правильных ответа, стрик 1
  - Флаг `isDemo: true` для идентификации

### 2. Новые шаги тура

#### Шаг 4 - 🔊 Произношение слова
```javascript
{
  target: null, // будет задан в onBefore
  title: '🔊 Произношение слова',
  text: 'На каждой карточке есть кнопка звука — нажми и услышишь произношение носителем языка.',
  onBefore() {
    // Переход на вкладку слов, схлопывание раскрытых карточек
    STEPS[4].target = document.querySelector('#words-grid .word-card .audio-btn');
  }
}
```

#### Шаг 5 - 📖 Пример и перевод
```javascript
{
  target: null,
  title: '📖 Пример и перевод',
  text: 'Нажми на карточку — она раскроется. Внутри увидишь пример использования, его перевод и кнопку прослушать пример.',
  onBefore() {
    // Автоматическое раскрытие карточки программно
    const card = document.querySelector('#words-grid .word-card');
    if (card && !card.classList.contains('expanded')) {
      card.click();
    }
    STEPS[5].target = document.querySelector('#words-grid .word-card');
  }
}
```

#### Шаг 10 - ⚙️ Настройки
```javascript
{
  target: () => document.querySelector('#user-avatar'),
  title: '⚙️ Настройки',
  text: 'Нажми на аватарку — там можно сменить голос озвучки, выбрать тему оформления и настроить лимит повторений в день.',
  onBefore() {
    // Закрытие дропдауна если открыт
    const dropdown = document.getElementById('user-dropdown');
    if (dropdown) dropdown.style.display = 'none';
    STEPS[10].target = document.querySelector('#user-avatar');
  }
}
```

### 3. Интеграция с жизненным циклом тура

#### При старте тура
```javascript
function buildTour() {
  addDemoWord(); // ← добавлено
  injectStyles();
  // ... остальной код
}
```

#### При завершении тура
```javascript
function endTour(completed) {
  // ... очистка DOM
  removeDemoWord(); // ← добавлено
  if (completed) {
    localStorage.setItem(TOUR_KEY, '1');
    // ... остальной код
  }
}
```

## 📋 Особенности реализации

- **Демо-слово существует только в памяти** - не сохраняется в localStorage/Supabase
- **Автоматическое раскрытие карточки** на шаге 5 через `card.click()`
- **Умное позиционирование** - шаг настроек подсвечивает аватарку без открытия модалки
- **Фолбэки** - все функции проверяют наличие `window.words` и `window.renderWords`
- **Индексация** - новые шаги корректно обновляют индексы последующих шагов

## 🎯 Результат

Тур теперь более информативен и показывает пользователям ключевые функции:
- Аудио произношение слов
- Примеры использования с переводом
- Доступ к настройкам приложения

Демо-слово гарантирует что у пользователя всегда будет слово для демонстрации функций.
