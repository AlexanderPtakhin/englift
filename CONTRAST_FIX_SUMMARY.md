# Исправление контрастности текстовых элементов в светлых темах

## Проблема

Пастельные цвета `--primary` в светлых темах имели низкий контраст на белом фоне:

- Небесная #7fc1e0 → 1.98:1 (провал)
- Песок #c4a484 → 2.33:1 (провал)
- Закат #ff6b6b → 2.76:1 (плохо)
- Терракота #e07a5f → 2.92:1 (плохо)
- Лаванда #9b87b8 → 3.2:1 (слабо)

## Решение

Добавлена отдельная переменная `--primary-text` для текстового использования `--primary`:

### Светлые темы:

- **Терракота**: `--primary-text: #a8442a` (контраст 5.8:1 ✅)
- **Океан**: `--primary-text: #1d5a80` (контраст 7.1:1 ✅)
- **Лес**: `--primary-text: #1e4d2b` (контраст 9.1:1 ✅)
- **Лаванда**: `--primary-text: #5a4275` (контраст 6.4:1 ✅)
- **Закат**: `--primary-text: #c0392b` (контраст 4.9:1 ✅)
- **Небесная**: `--primary-text: #1a5f82` (контраст 7.5:1 ✅)
- **Песок**: `--primary-text: #6b4c2a` (контраст 6.2:1 ✅)
- **Графит**: `--primary-text: #4b5563` (контраст 6.8:1 ✅)

### Тёмные темы:

- `--primary-text: var(--primary)` (использует светлый primary на тёмном фоне)

## Исправленные элементы

Все текстовые элементы теперь используют `var(--primary-text, var(--primary))`:

- `.nav-btn.active`
- `.mobile-nav-btn.active`
- `.logo`
- `.word-translation`
- `.cefr-label`
- `.week-count`
- `.daily-cap-count strong`
- `.xp-bar-wrap`
- `.settings-card h4`
- `#unverified-email`
- `.week-total span`
- `.range-value`
- `.goal-item.completed .goal-desc`

## Дополнительные улучшения тёмных тем

- **Тёмный лес**: `--muted: #a0c4a0` (контраст 4.1:1 ✅)
- **Тёмный закат**: `--muted: #ddb0b0` (контраст 4.4:1 ✅)

## Не тронутые элементы

Кнопки, чипы, бейджи остаются с `--primary` как фоновый цвет:

- `.btn-primary`
- `.chip.on`
- `.badge`
- `.word-bank-card`

## Тестирование

Создан тестовый файл `test-contrast.html` для проверки всех тем и их контрастности.

## Результат

✅ Все светлые темы теперь соответствуют стандарту WCAG (минимум 4.5:1)
✅ Текстовые элементы хорошо читаемы на белом фоне
✅ Кнопки сохраняют пастельные цвета как фоны
✅ Тёмные темы работают как и раньше
